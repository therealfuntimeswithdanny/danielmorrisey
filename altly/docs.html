Alt Text Generator API

Programmatically generate descriptive alt text for images using Cloudflare Workers AI.

Endpoint

POST https://alt-text.madebydannyuk.workers.dev/api

The API accepts an image file as multipart/form-data and returns a JSON object with the image URL and generated alt text.

Example — cURL

curl -X POST \ 
  -F "file=@your-image.png" \
  https://alt-text.madebydannyuk.workers.dev/api

Example — JavaScript Fetch

const form = new FormData();
form.append("file", fileInput.files[0]);

const res = await fetch("https://alt-text.madebydannyuk.workers.dev/api", {
  method: "POST",
  body: form
});

const data = await res.json();
console.log(data.alt);

Response Format

{
  "success": true,
  "url": "https://public-cdn.madebydanny.uk/alt-text/uuid-image.png",
  "alt": "A white cat sitting on a windowsill looking outside."
}

CORS Configuration

Add this to your Worker to allow public API access:

const corsHeaders = {
  "Access-Control-Allow-Origin": "*",
  "Access-Control-Allow-Methods": "GET, POST, OPTIONS",
  "Access-Control-Allow-Headers": "Content-Type",
};

if (request.method === "OPTIONS") {
  return new Response(null, { headers: corsHeaders });
}

// Example return with CORS:
return Response.json({ success: true, url, alt }, { headers: corsHeaders });
